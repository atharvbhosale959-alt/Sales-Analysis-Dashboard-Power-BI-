# Sales-Analysis-Dashboard-Power-BI-
📊 Sales Analysis Dashboard (Power BI)

🚀 Project Overview

This project showcases an interactive Sales Analysis Dashboard built using Power BI after 1 month of learning and hands-on practice. The goal is to analyze sales data and generate meaningful business insights.

🎯 Objectives
Analyze overall sales performance
Identify top-performing products
Understand regional and category-wise sales trends
Create an interactive and user-friendly dashboard
🛠️ Tools & Technologies
Power BI
Microsoft Excel
DAX (Data Analysis Expressions)
📁 Dataset

The dataset includes:

Order ID
Order Date & Ship Date
Customer Name & Segment
Product Name, Category, Sub-Category
Region
Sales
🔄 Data Processing
Cleaned and transformed raw data
Handled missing values and duplicates
Created calculated measures (Total Sales, Product-wise Sales)
Built data model for better analysis
📊 Dashboard Features
📌 Total Sales KPI
📌 Sales by Product Name
📌 Category & Sub-Category Analysis
📌 Region-wise Sales
📌 Monthly Sales Trend
💡 Key Insights
Top products contributing to revenue
High-performing regions
Sales trends over time
Category-wise performance

📌 Outcome

This project demonstrates how data can be transformed into actionable insights using Power BI, helping businesses make data-driven decisions.
